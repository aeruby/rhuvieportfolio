# Rhuvie Joyce Amarille - Professional Portfolio

This repository hosts the source code for my personal portfolio website, showcasing my work as a UI/UX & Web Designer and Computer Science student. It features a responsive layout, custom CSS animations, and details about my academic journey, projects, skills, and certifications.

## 🚀 Features

- **Typewriter Animation**: Dynamic introductory text built using custom CSS keyframe animations.
- **Responsive Navigation**: Includes a functional dropdown mobile menu for easy navigation on smaller viewports.
- **Project Catalog**: Categorized sections showcasing developed web applications, UI/UX wireframes, and design infographics.
- **Certifications & Awards Showcase**: Highlighting key academic and professional achievements.
- **Responsive Layout**: Styled using custom CSS Grid and Flexbox layouts for mobile, tablet, and desktop viewports.

---

## 🛠️ Built With

- **HTML5**: Semantic markup structuring the content.
- **CSS3**: Custom layouts, responsive variables, typography, and interactive hover effects.
- **Vanilla JavaScript**: Lightweight script handling navigation toggles and mobile menu animations.
- **FontAwesome**: Scalable vector icons for professional presentation.
